## 完成の定義

私は「完成」を、その主題の基礎知識を持つ人が容易に理解できる、明確で簡潔な説明があることと定義します。

### 目次

- [0. 共通](#0-共通)
- [1. 機械学習](#1-コンピュータサイエンス--機械学習)


### 0. 共通
- [x] 理解を支える具体的な文章を記述する（単に「理解しました...」と書くだけではない）
- [x] 適用できる条件を記述する（誰が、いつ、どこで）
- [x] ライセンスや著作権に関する情報を含める

### 1. コンピュータサイエンス / 機械学習

- [x] 明確な入力と出力
- [x] 擬似コードでアルゴリズムを記述する
- [x] 使用するデータセットを説明する
- [x] 明確な計算順序
- [x] 類似アルゴリズムとの違いを記述する
