# Machine Learning

This directory is dedicated to machine learning reseach papers, implementations, and resources.
It serves as a repository for organizing and sharing knowledge in the field of machine learning.

## Directory Structure

- `YYYY/`: When publishing a new paper (or books and so on), create a new directory named after the year of publication (e.g., `2023/`, `2024/`).
  - `paper-title/`: Inside the year directory, create a subdirectory named after the paper title (use hyphens for spaces).
    - `README.md`: Summary of the paper, including key findings and contributions.
    - `code/`: (Optional) Directory containing code implementations related to the paper.
    - `data/`: (Optional) Directory for datasets used or generated in the paper.
    - `figures/`: (Optional) Directory for figures and visualizations from the paper
    - `notes.md`: (Optional) Personal notes or insights about the paper.
