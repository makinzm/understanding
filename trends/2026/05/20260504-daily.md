   - Why I recommend: Covers concrete permission-bypass and prompt-injection attack paths that emerge when AI is integrated into CI/CD pipelines — immediately actionable for any team using AI-assisted coding in a GitHub Actions workflow.

3. [Supply Chain Campaign Targets SAP npm Packages with Credential-Stealing Malware](https://www.wiz.io/blog/mini-shai-hulud-supply-chain-sap-npm)
   - Why I recommend: Documents a live supply chain attack targeting npm packages used by SAP; the "Mini Shai Hulud" campaign shows how credential-stealing malware is evolving, relevant to any engineer managing npm dependencies.

4. [Securing GitHub: Wiz Research uncovers Remote Code Execution in GitHub.com and GitHub Enterprise Server (CVE-2026-3854)](https://www.wiz.io/blog/github-rce-vulnerability-cve-2026-3854)
   - Why I recommend: Critical RCE in GitHub's own git infrastructure is extremely high impact — every team hosting ML code and pipelines on GitHub should understand this class of vulnerability.

5. [opy Fail: Universal Linux Local Privilege Escalation Vulnerability](https://www.wiz.io/blog/copyfail-cve-2026-31431-linux-privilege-escalation-vulnerability)
   - Why I recommend: CVE-2026-31431 allows any unprivileged local user to escalate to root on Linux — critical to patch in ML training clusters and cloud VMs where many users share the same host.

6. [Why are neural networks and cryptographic ciphers so similar? (2025)](https://reiner.org/neural-net-ciphers)
   - Why I recommend: Draws a surprising structural analogy between neural network forward passes and symmetric-key cipher rounds; thought-provoking for ML researchers interested in the mathematical foundations of deep learning.

7. [How OpenAI delivers low-latency voice AI at scale](https://openai.com/index/delivering-low-latency-voice-ai-at-scale/)
   - Why I recommend: Engineering deep-dive into real-time inference infrastructure for voice models, including latency optimization techniques applicable to any low-latency ML serving system.

8. [Let's talk about LLMs](https://www.b-list.org/weblog/2026/apr/09/llms/)
   - Why I recommend: A candid, skeptical assessment of where LLMs stand — useful for calibrating expectations and preparing grounded talking points when discussing AI with non-ML colleagues.

9. [Microsoft Edge stores all passwords in memory in clear text, even when unused](https://twitter.com/L1v1ng0ffTh3L4N/status/2051308329880719730)
   - Why I recommend: Striking security regression in a major browser; directly relevant to any developer or team storing credentials on dev machines, and a good case study in defensive memory handling.

10. [Sierra Raises $950M at $15B Valuation](https://sierra.ai/blog/better-customer-experiences-built-on-sierra)
