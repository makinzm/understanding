4. [Introducing the Wiz Red Agent- AI-Powered Attacker](https://www.wiz.io/blog/introducing-the-wiz-red-agent)
   - Why I recommend: Wiz launched an AI agent that performs automated red-teaming of cloud environments — a signal that autonomous security testing agents are becoming productized and mainstream, relevant both for understanding attack surfaces and for building defensive tooling.

5. [StepFun 3.5 Flash is #1 cost-effective model for OpenClaw tasks (300 battles)](https://app.uniclaw.ai/arena?tab=costEffectiveness&via=hn)
   - Why I recommend: A new benchmark showing StepFun 3.5 Flash leading on cost-efficiency for coding/reasoning tasks across 300 head-to-head battles — useful data for ML engineers choosing models for production pipelines where cost matters.

6. [Show HN: Real-time dashboard for Claude Code agent teams](https://github.com/simple10/agents-observe)
   - Why I recommend: Observability tooling for Claude Code agentic teams is emerging — this dashboard tracks multi-agent activity in real time, directly applicable to teams already using Claude Code for automated workflows like this one.

7. [Introducing Wiz Agents & Workflows: Security at the Speed of AI](https://www.wiz.io/blog/introducing-wiz-agents)
   - Why I recommend: Wiz is building a full agentic security platform (Red Agent, Blue Agent, Green Agent + Workflows) announced at their conference. Understanding how AI agents are being applied in the security domain is highly relevant for ML engineers building similar agentic systems.

8. [Would You Click 'Accept'? Automatically detecting malicious Azure OAuth applications using LLMs](https://www.wiz.io/blog/detecting-malicious-oauth-applications)
   - Why I recommend: A practical application of LLMs to classify malicious OAuth consent prompts at scale — a concrete security use case for ML classification that is directly actionable and well-documented by Wiz Research.

9. [Is BGP safe yet?](https://isbgpsafeyet.com/)
   - Why I recommend: A live dashboard tracking BGP security (RPKI deployment) across major ISPs — BGP hijacking remains a real infrastructure threat, and this site provides a quick, current snapshot of which networks are still vulnerable.

10. [Show HN: Zerobox – Sandbox any command with file, network, credential controls](https://github.com/afshinm/zerobox)
    - Why I recommend: A lightweight sandboxing tool for running untrusted commands with fine-grained controls over file system, network, and credential access — practical for ML engineers running model inference pipelines or third-party scripts safely.
